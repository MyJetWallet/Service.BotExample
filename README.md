# How to run Mac

Add in home directory file with name ".myjetwallet" and content:

```
BotExample:
  SeqServiceUrl:
  ZipkinUrl:
  ElkLogs:
    - Urls
    - User
    - Password
    - IndexPrefix: jet-logs-uat
```


# How to run Windows

Add in roo directory of your disk file with name ".myjetwallet" and content:
